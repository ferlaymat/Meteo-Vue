# Meteo-Vue

This project is a sample weather app to show basic cases in vue.js

## Requirements

- Node V24.13.1

## Getting Started

Firstly, create an account to get the api key and replace it in Constants.ts. It is free and allows 1000 calls per day: openweathermap.org

then launch the front with:

```bash
cd ./meteo-app
npm run dev
```

## Projects and URLs

| Service | URL                   | Use case                                             |
| ------- | --------------------- | ---------------------------------------------------- |
